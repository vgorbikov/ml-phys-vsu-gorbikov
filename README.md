# Лабораторные работы по курсу "Аналитика больших объёмов данных"

## Рабочая среда

Большинство примеров будут реализованы в [Jupyter Notebook](https://jupyter.org/) в [GoogleColab](https://colab.research.google.com/?hl=ru_RU).

Библиотеки, которые будут использоваться в течение курса:
- [NumPy](https://numpy.org/), [SciPy](https://scipy.org/)
- [Matplotlib](https://matplotlib.org/)
- [Pandas](https://pandas.pydata.org/)
- [scikit-learn](https://scikit-learn.org/stable/)
